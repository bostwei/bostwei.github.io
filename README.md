# shiyan.github.io
Personal website for Shiyan Wei.

## About
This is a personal academic website hosted on GitHub Pages, featuring sections for education, research interests, publications, and contact information.

## Viewing the Website
Once GitHub Pages is enabled, the website will be available at: `https://bostwei.github.io/`

## Customization
To personalize the website, edit the following files:

### index.html
- Update your name, title, and bio in the About section
- Add your education details (degrees, universities, years)
- List your research interests
- Add your publications
- Update contact information and social media links
- Replace the placeholder image URL with your own photo

### style.css
- Modify colors by changing the color values (e.g., `#2c3e50`, `#3498db`)
- Adjust fonts, spacing, and layout as needed

## GitHub Pages Setup
To enable GitHub Pages:
1. Go to your repository Settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select the branch you want to deploy (e.g., `main` or `copilot/create-personal-website-github`)
4. Click "Save"
5. Your website will be published at `https://bostwei.github.io/`

## Features
- Responsive design (works on desktop, tablet, and mobile)
- Clean, professional academic layout
- Smooth navigation with anchor links
- Easy to customize and maintain
- No build process required - pure HTML/CSS
